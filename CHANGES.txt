RS232_WEB_CLOUD_002_Render_MainUI

Cloud-only changes:
- New Main-style read-only monitoring page.
- Added familiar buttons: READ/REFRESH, ADD, HOLD, NO+/NO-, EXPORT CSV, PRINT, CLEAR LOCAL.
- Added locked visual buttons for LOOP START, LOOP STOP, SETTINGS.
- Locked buttons do not call ESP32 and only show allow_remote_commands=0 warning.
- Removed/omitted FFT, Settings, Diagnostics, SD Tools, SD file access.
- POST /api/push remains protected by DEVICE_TOKEN.
- GET / and JSON read endpoints remain optionally protected by VIEW_TOKEN.
- /api/read, /api/loop, /api/command, /api/settings and /sdtools return 403 Forbidden.
- History remains RAM-only.
- RS232_WEB_108 firmware is not touched.
